Resumen de la asignatura

En esta seccion, se mostrara el programa de la asignatura, mostrando los diferentes temas agrupados

PARTE II. Bloque IV. INTRODUCCIÓN AL DESARROLLO WEB. Introducción y manejo de repositorios de SW

Unidad_1_Introducción al desarrollo de software

Este tema aborda los conceptos fundamentales relacionados con la creacion de programas informaticos. En este tema, se exploran los principios basicos del desarrollo de software, incluyendo conceptos como algoritmos, lenguajes de programacion, paradigmas de desarrollo y el ciclo de vida del software.

Unidad 2_Herramientas de desarrollo

Durante este tema, se examinan diversas herramientas que facilitan la creación, prueba y mantenimiento de aplicaciones informáticas. Esto incluye entornos de desarrollo integrados (IDE), sistemas de control de versiones, compiladores, depuradores y otros recursos que optimizan la eficiencia y calidad del trabajo de los desarrolladores.

PARTE II. Bloque V. INTRODUCCIÓN AL DESARROLLO WEB. CSS

Unidad_1_CSS I

2023 - 2024

Se introduce el lenguaje de estilo Cascading Style Sheets (CSS). Se exploran los conceptos básicos de CSS, incluyendo selectores, propiedades y valores, que permiten dar estilo y diseño a páginas web.

Unidad_2_CSS II

2023 - 2024

Profundiza en las capacidades avanzadas de Cascading Style Sheets. Se examinan temas más complejos, como el diseño responsivo para adaptar la presentación a diferentes dispositivos, el uso de animaciones y transiciones para mejorar la experiencia del usuario, así como la organización eficiente del código CSS mediante conceptos como la especificidad y la modularidad.

PARTE II. Bloque V. INTRODUCCIÓN AL DESARROLLO WEB. Creación de páginas Web HTML5

Unidad_1_Introducción al desarrollo web

2023 - 2024

Se proporciona una visión general de los principios y conceptos fundamentales relacionados con la creación de aplicaciones y sitios web. Se exploran aspectos clave como la arquitectura cliente-servidor, los protocolos web, y las tecnologías esenciales como HTML.

Unidad_2_HTML

2023 - 2024

Se enfoca en el lenguaje de marcado fundamental para la creación de páginas web. Durante esta unidad, se exploran los elementos y atributos clave de HTML (Hypertext Markup Language). Los estudiantes aprenden a estructurar el contenido de una página web mediante etiquetas como encabezados, párrafos, listas y enlaces. También se abordan temas avanzados, como formularios y multimedia, permitiendo a los desarrolladores comprender cómo construir páginas web efectivas y ricas en contenido utilizando HTML.
